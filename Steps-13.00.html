Adding Measures 13

These steps will walk us through creating all the measures we'll use in this report.

All measures will be placed in the Measures Folder we created earlier, to make things simple and organize. 

Cautions: when creating the measures make sure you're in the 'Data View' so

that you can see the table columns and compare to the formula/measures column name to avoid any issues and errors. 

  1. From the Fields pane select the 'Measures Folder' then, 
  2. in the ribbon, select New Measure
  3. A formula bar will appear like so, 
  4. Copy one of the measures provided below and paste into the formula bar
  5. Press Enter or the Check mark to the left of the formula bar to create/commit the formula
  6. After, the measure will appear in the left Field pane, under the Measure Folder
  7. Select it and change/format the Data type according to the formatting instructions provided below. 

Example: TotalShooting measure


Formatting instructions:

  1. Shooting per day: Decimal (2 dec places)
  2. All % and YoY measures will (percentages, 2 dec place) 
  3. All other will be Whole Number. 

Measures/Formulas

Total Shooting = DISTINCTCOUNT(Shooting[id])

 

Shooting LY = CALCULATE (

Total Shooting,

SAMEPERIODLASTYEAR('Date'[Date]) )

 

Shooting Per Day = CALCULATE (

AVERAGEX (VALUES('Date'[Date]), [Total Shooting]),

CALCULATETABLE (‘Date’, 'Date'[DateWithTransaction] = true ) )

 

Shooting YoY % = CALCULATE (

 DIVIDE (([Total Shooting] - [Shooting LY]), [Shooting LY], 0 ),

 CALCULATETABLE (‘Date’, 'Date'[DateWithTransaction] = true))

 

Total Threats = SUM(Shooting [Threats_Level]  )

 

Threats LY = CALCULATE (Total Threats,

SAMEPERIODLASTYEAR('Date'[Date]))

 

Threats % = [Total Threats] / COUNTROWS(Shooting)

 

 

Threat YoY = CALCULATE (

                DIVIDE (([TotalThreats] - [Threats LY]) , [Threats LY], 0 ),

                CALCULATETABLE ('Date', 'Date'[DateWithTransaction] = true ) )

 

Mental Health = SUM (Shooting [Mental_Health])

 

Mental Health LY = CALCULATE ([Mental Health],

SAMEPERIODLASTYEAR('Date'[Date]))

 

[Mental Health % ] = [Total Mental Health] / COUNTROWS(Shooting)

 

Mental Health YoY = CALCULATE (

DIVIDE (([Mental Health]  -  [Mental Health LY]), [Mental Health LY], 0 ),

CALCULATETABLE (‘Date', 'Date'[DateWithTransaction] = true ))

 

Flee = SUM (Shooting [Flee])

 

Flee LY = CALCULATE([Flee],

SAMEPERIODLASTYEAR('Date'[Date]))

 

Flee % = [Flee] / COUNTROWS(Shooting)

 

Flee YoY = CALCULATE (

DIVIDE(([Flee] - [Flee LY]), [Flee LY], 0) ,

CALCULATETABLE ('Date', 'Date'[DateWithTransaction] = true ))

 

TotalCamNUsed = CALCULATE(COUNT(Shooting [BodyCamState]) , Shooting[BodyCamState] = 0 )

 

TCamNU LY = CALCULATE([TotalCamNUsed],

SAMEPERIODLASTYEAR('Date'[Date]))

 

Cam NUsed % =   [TotalCamNUsed]  /COUNTROWS( [Shooting] )

 

CamNU YoY = CALCULATE (

                                                DIVIDE (( [TotalCamNUsed] - [TCamNU LY] ), [TCamNU LY], 0 ),

                                                CALCULATETABLE ( 'Date', 'Date'[DateWithTransaction] = TRUE ))
