Add Line and Clustered Column Chart 16

Steps: Line and Stacked Column Charts

  1. Make sure no visual is selected in the canvas then in the Visualizations pane click on the "Line and Stacked Column Chart" to add it to the canvas. 
  2. Under Visualizations drag Year, Month, Day into the X-axis from the Date table 
  3. Drag the 'Shooting Per Day' measure in the Column y-axis 
  4. Add Shooting YoY, Flee YoY, CamNU YoY, Mental Health YoY, and Threat YoY measures into the Line y-axis field 
  5. Add all 6 KPIs in the Tooltips field, we'll repeat this steps for all of the visuals. 

Format:

  1. Click on  and under Visual in X-axis change the Values: make Font = Segoe UI, Size = 9, Color = Black, Title = off
  2. Make Y-axis and the Secondary y-axis the same. Also, in Secondary axis make Align zeroes = on. 
  3. Then in Gridlines make Horizontal Style = Dotted, Color = Light Grey, Width = 1px. Do the same for Vertical Gridlines.
  4. Leve the Legend on, under Columns the Color = #118DFF
  5. under Lines make the Shape = Solid, Join Type = Round, Stroke = 2px, and Shade area = On. 
  6. in Colors make Threats = #E044A7, MH = #6B007B, CamNU = #E66C37, Flee = #12239E, Shooting = #e8d166
  7. Make Markers, Data labels, and Series labels = off
  8. In General setting: Make Height = 284, Width = 622, Horizontal = 0, Vertical 148, make Title = "Year", Heading = 3, Fond = DIN, Size = 14, Color = Black, in Background make Color = White, Transparency = 85, Visual border = off,
