Data Tranform 9


This steps will focus on cleaning the data. Removing bad data, add headers, etc...

Steps: (Shooting Table)

  1. Remove unwanted columns,
  2. From the Home tab click Transform data, and from the ribbon click Chose Columns 
  3. Unselect: name, armed, arms_category, manner_of_death columns and click OK




Steps: (Change Values into 1 & 0)

  1. we will change the sign_of_mental_illnes, Threat_level, Flee, and body_camera values
  2. from the Home tab click Transform data then on the Queries pane make sure Shooting table is selected. 
  3. select 'sign_of_mental_illnes' column then change the Data Type to Text in the ribbon
  4. Right click on TRUE under the column then select Replace Values, in the new Replace Values window enter 1 in the Replace With.
  5. then click on FALSE and replace the values with 0.


Follow the above step to change: 

Threats, Flee, and body_camera values as follow:

  1. threat_level ( attack =1, other = 1, undetermine  0)
  2. flee ( foot =1, car =1, other =1, not fleeing = 0)
  3. body_camera (true = 1, false =0)

Remember to change the Data Type to avoid any errors. 


Steps: (Region Table) 

  1. these steps will promote the first row into header
  2. from the Home tab click Transform data then click on Use First Row as Headers from the ribbon
